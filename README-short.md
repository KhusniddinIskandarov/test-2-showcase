# 🌿 Plants Website

🚀 A modern, responsive, and performance-optimized plant shop website built with clean frontend architecture and scalable structure.

---

## 🔗 Live Demo

👉 Live Demo >---> [plants7.vercel.app](https://plants7.vercel.app) 🔗

---

## 🧠 About The Project

This project focuses on building a **high-performance frontend application** using modern best practices.

It is designed with a strong emphasis on:

- ⚡ Performance optimization (image optimization, lazy loading)
- 🧩 Modular and maintainable architecture
- 🎯 Clean and user-friendly UI/UX
- 🚀 Production-ready deployment

---

## 🛠️ Tech Stack

- HTML5
- TailwindCSS + Custom CSS
- Vanilla JavaScript (ES6+)
- Vite

---

## ✨ Features

- ⚡ Fast and optimized performance
- 📱 Fully responsive design (Mobile / Tablet / Desktop)
- 🎯 Clean UI/UX
- 🧩 Modular JavaScript structure
- 🖼️ Optimized images (WebP + lazy loading)
- 🎞️ Interactive UI elements (Swiper slider)
